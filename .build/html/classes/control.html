Control¶

Note

Control is derived from Widget class.

This class is an abstract base class for objects called controls. The purpose is to define properties and methods shared for derivated classes.

Constructor¶

Control class has a single constructor:

Control:new ( string sControlName, object oParent, number nPosX, number nPosY, number nWidth, number nHeight, number nID )

Arguments¶

These arguments are received by class constructor.
